# To Start The project

Installez XAMPP (MySQL).

Créez la base hospital_db.

Créez votre environnement virtuel : python -m venv .venv

Activez-le : .\.venv\Scripts\activate

Installez les dépendances : pip install mysql-connector-python pandas anthropic python-dotenv google-generativeai

Exécutez sql-schema-complete.sql sur votre base hospital_db.

Exécutez run_validation.sql sur votre base hospital_db.

Workflow (À chaque exécution) :

Étape 1 : Exécutez python extract_to_staging_db.py

Étape 2 : Exécutez python process_staging_to_production.py

Étape 3 : Exécutez python export_to_csv.py

Résultat : Le dossier /csv_export contient vos fichiers propres.
